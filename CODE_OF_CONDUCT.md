# Code of Conduct

## Our Standards

Mortise is an open source project. We expect all participants to:

- Be respectful and constructive in all interactions
- Accept feedback gracefully
- Focus on what is best for the community and project
- Show empathy toward other contributors

Unacceptable behavior includes harassment, personal attacks, discriminatory language, and any conduct that would be inappropriate in a professional setting.

## Enforcement

Maintainers have the right to remove, edit, or reject contributions, comments, and other interactions that do not align with this code of conduct, and to ban contributors for repeated or severe violations.

## Reporting

Report issues by emailing the maintainers directly or opening a private GitHub security advisory. Reports will be handled confidentially.
